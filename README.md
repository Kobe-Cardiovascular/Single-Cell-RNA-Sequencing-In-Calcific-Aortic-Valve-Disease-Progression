# Single-Cell-RNA-Sequencing-In-Calcific-Aortic-Valve-Disease-Progression

We analysed single-cell RNA sequencing data from four patients with non-calcified aortic regurgitation (AR) and 12 patients with aortic stenosis (AS) who underwent surgical aortic valve replacement at Kobe University Hospital.

The code is written in R and uses the Seurat package.
